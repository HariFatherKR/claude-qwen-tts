# Sample Voice Files

This directory contains sample reference voices for TTS.

## Files

- `ko_male.wav` - Korean male voice sample
- `ko_male.txt` - Transcript for male sample
- `ko_female.wav` - Korean female voice sample
- `ko_female.txt` - Transcript for female sample

## Adding Your Own Voice

1. Record a 5-10 second audio clip
2. Save as WAV format (recommended: 24kHz, mono)
3. Run `/tts-init` to register your voice

## Tips for Good Reference Audio

- Use a quiet environment
- Speak naturally and clearly
- Avoid background music or noise
- 5-10 seconds is optimal length
- Match the transcript exactly to what was spoken
